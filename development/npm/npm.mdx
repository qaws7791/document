## NPM 버전

https://docs.npmjs.com/about-semantic-versioning/

4.14.18

major.minor.patch

1.0.0부터 시작

major: 호환 불가능

minor: 호환 가능 / 새로운 기능

patch: 호환 가능 / 버그 수정



기호 없음: 지정된 버전만 사용

"^"기호: 메이저 버전 안에서 가장 최신 버전 사용 (메이저 버전은 유지)

"~" 기호: 마이너 버전 안에서 가장 최신 버전 사용 (마이너 버전은 유지)
