### 1948년

- Claude Shannon이 [정보 이론](https://en.wikipedia.org/wiki/A_Mathematical_Theory_of_Communication)을 발표

## 1960년대


### 1965년

- National Physical Laboratory의 Donald Davies가 [패킷 교환](https://en.wikipedia.org/wiki/Packet_switching)을 발표


### 1969년

- 미국 국방부의 [ARPANET](https://en.wikipedia.org/wiki/ARPANET)이 최초로 연결됨


## 1970년대


### 1974년

- Vint Cerf와 Bob Kahn이 [TCP/IP](https://en.wikipedia.org/wiki/Transmission_Control_Protocol)를 발표


## 1980년대


### 1980년

- [CERN](https://en.wikipedia.org/wiki/CERN)의 [Tim Berners-Lee](https://en.wikipedia.org/wiki/Tim_Berners-Lee)가 [ENQUIRE](https://en.wikipedia.org/wiki/ENQUIRE)를 개발


### 1983년

- [Paul Mockapetris](https://en.wikipedia.org/wiki/Paul_Mockapetris)가 RFC 882, RFC 883을 통해 [DNS(Domain Name System)](https://en.wikipedia.org/wiki/Domain_Name_System)을 발표


### 1989년


## 1990년대


### 1990년

- [Tim Berners-Lee](https://en.wikipedia.org/wiki/Tim_Berners-Lee)가 [WorldWideWeb](https://en.wikipedia.org/wiki/WorldWideWeb)을 개발. 이는 최초의 웹 브라우저이자 편집기.


### 1993년

- [Tim Berners-Lee](https://en.wikipedia.org/wiki/Tim_Berners-Lee)가 [HTML(HyperText Markup Language)](https://en.wikipedia.org/wiki/HTML)의 첫 번째 버전을 발표

- NCSA의 [Marc Andreessen](https://en.wikipedia.org/wiki/Marc_Andreessen)와 [Eric Bina](https://en.wikipedia.org/wiki/Eric_Bina)가 [Mosaic](https://en.wikipedia.org/wiki/Mosaic_(web_browser)) 브라우저를 개발


### 1994년

- Netscape의 [Marc Andreessen](https://en.wikipedia.org/wiki/Marc_Andreessen)와 [Eric Bina](https://en.wikipedia.org/wiki/Eric_Bina)가 [Netscape Navigator](https://en.wikipedia.org/wiki/Netscape_Navigator) 브라우저를 개발


### 1995년

- RFC 1866을 통해 [HTML(HyperText Markup Language)](https://en.wikipedia.org/wiki/HTML)의 두 번째 버전인 HTML 2.0을 발표

- Netscape의 [Brendan Eich](https://en.wikipedia.org/wiki/Brendan_Eich)가 [JavaScript](https://en.wikipedia.org/wiki/JavaScript)를 개발

- Microsoft의 [Thomas Reardon](https://en.wikipedia.org/wiki/Thomas_Reardon)가 [Internet Explorer](https://en.wikipedia.org/wiki/Internet_Explorer) 브라우저를 개발

- Opera의 [Jon Stephenson von Tetzchner](https://en.wikipedia.org/wiki/Jon_Stephenson_von_Tetzchner)와 [Geir Ivarsøy](https://en.wikipedia.org/wiki/Geir_Ivars%C3%B8y)가 [Opera](https://en.wikipedia.org/wiki/Opera_(web_browser)) 브라우저를 개발


### 1996년

- [Wium Lie](https://en.wikipedia.org/wiki/H%C3%A5kon_Wium_Lie)와 [Bert Bos](https://en.wikipedia.org/wiki/Bert_Bos)가 [CSS(Cascading Style Sheets)](https://en.wikipedia.org/wiki/CSS)의 첫 번째 버전을 발표


### 1997년

- [W3C](https://en.wikipedia.org/wiki/World_Wide_Web_Consortium) 권장 사항을 통해 [HTML(HyperText Markup Language)](https://en.wikipedia.org/wiki/HTML)의 세 번째 버전인 HTML 3.2(1월), HTML 4.0(12월)을 발표

- [Ecma International](https://en.wikipedia.org/wiki/Ecma_International)가 [JavaScript](https://en.wikipedia.org/wiki/JavaScript)의 표준화를 위해 [ECMAScript](https://en.wikipedia.org/wiki/ECMAScript)를 발표


### 1998년

- [W3C](https://en.wikipedia.org/wiki/World_Wide_Web_Consortium) 권장 사항을 통해 [CSS(Cascading Style Sheets)](https://en.wikipedia.org/wiki/CSS)의 두 번째 버전인 CSS 2.0을 발표


### 1999년

- 첫 번째 3개의 CSS 3 초안(Color Profiles, Multi-column layout, and Paged Media)을 발표

- [Internet Explorer 5.0](https://en.wikipedia.org/wiki/Internet_Explorer_5)을 통해 [XMLHttpRequest](https://en.wikipedia.org/wiki/XMLHttpRequest)가 처음으로 구현되어 비동기적 데이터 전송([Ajax](https://en.wikipedia.org/wiki/Ajax_(programming)))을 가능하게 함


## 2000년대


### 2003년

- [Apple](https://en.wikipedia.org/wiki/Apple_Inc.)이 [Safari](https://en.wikipedia.org/wiki/Safari_(web_browser)) 브라우저를 개발


### 2004년

- [Mozilla Foundation](https://en.wikipedia.org/wiki/Mozilla_Foundation)가 [Mozilla Firefox](https://en.wikipedia.org/wiki/Firefox) 브라우저를 개발


### 2006년

- [John Resig](https://en.wikipedia.org/wiki/John_Resig)가 [jQuery](https://en.wikipedia.org/wiki/JQuery) 라이브러리를 개발

- [W3C](https://en.wikipedia.org/wiki/World_Wide_Web_Consortium)는 [Ajax](https://en.wikipedia.org/wiki/Ajax_(programming))에 대한 표준을 위해 [XMLHttpRequest](https://en.wikipedia.org/wiki/XMLHttpRequest)의 첫 번째 사양을 발표


### 2008년

- [Google](https://en.wikipedia.org/wiki/Google)이 [V8 자바스크립트 엔진](https://en.wikipedia.org/wiki/V8_(JavaScript_engine))을 탑재한
[Google Chrome](https://en.wikipedia.org/wiki/Google_Chrome) 브라우저를 개발


## 2010년대


### 2011년

- [W3C](https://en.wikipedia.org/wiki/World_Wide_Web_Consortium) 권장 사항을 통해 CSS 2의 개정판인 CSS 2.1을 발표


### 2014년

- [W3C](https://en.wikipedia.org/wiki/World_Wide_Web_Consortium) 권장 사항을 통해 [HTML(HyperText Markup Language)](https://en.wikipedia.org/wiki/HTML)의 다섯 번째 버전인 HTML 5.0을 발표


## 2020년대


### 2020년

- [Microsoft](https://en.wikipedia.org/wiki/Microsoft)이 [Chromium](https://en.wikipedia.org/wiki/Chromium_(web_browser))기반의 새로운 [Microsoft Edge](https://en.wikipedia.org/wiki/Microsoft_Edge) 브라우저를 개발


- 1990년 최초의 웹이 팀 버너스리에 의해 탄생
- 초기 웹은 **SSG(Static Site Generation)**, **MPA(Multi Page Application)** 방식
- 1995년 `JavaScript` 발표
- 1996년 `CSS(Cascading Style Sheets)` 발표
- 서버 사이드 스크립트
  - 1995년 `PHP(Hypertext Preprocessor)` 발표
  - 1996년 `ASP(Active Server Pages)` 출시
  - 1999년 `JSP(Jakarta Server Pages)` 발표
- 서버 사이드 스크립트 언어를 통해 서버에서 동적으로 HTML을 생성하는 **SSR(Server Side Rendering)** 사용
- 2000년 `XMLHttpRequest` 고안. 비동기적 데이터 전송을 통한 부분적 페이지 업데이트
- 2006년 구글이 Gmail, Google Maps 등에 `Ajax`를 사용하기 시작하면서 Ajax 기술과 SPA가 주목을 받기 시작함.
- Ajax를 사용한 **CSR(Client Side Rendering)** 및 **SPA(Single Page Application)**의 등장
- 웹이 보편화되면서 서버와 클라이언트 각각의 개발에 대한 복잡도 증가로 인해 백엔드와 프론트엔드가 분리
- 2010년대 React, Angular, Vue 등 컴포넌트 기반의 개발이 유행
- 2015년 CSR 단점인 `느린 초기 로딩 속도`, `SEO 최적화` 등에서의 문제로 인해 Jekyll, Gatsby와 같은 SSG가 인기를 얻기 시작
- 2010년대 후반 Next.js, Nuxt.js와 같은 SSR 프레임워크의 부상
- 2020년 Next.js에서 SSG의 장점을 유지하면서도 동적 데이터 업데이트가 가능한 **ISR(Incremental Static Regeneration)** 렌더링 방식을 선보임
