## 휴리스틱 평가란

휴리스틱 평가는 사용자 경험 디자인에서 사용자 인터페이스를 평가하는 방법 중 하나로, 전문가가 제품을 사용하고 제품의 사용자 경험을 평가하는 방법이다.

## 제이콥 닐슨의 휴리스틱 평가 10가지

### 1. 시스템 상태 시각화

사용자에게 시스템의 현재 상태를 시각적으로 보여주는 것

ex. 브레드 크럼으로 현재 페이지 위치, 로딩 상태

### 2. 현실 세계를 반영한 시스템

사용자가 일상에서 접하는 것과 가깝게 표현하여 친숙함을 주는 방법

ex. 버튼 및 폼 양식,

### 3. 사용자의 제어와 자유

사용자에게 적절한 통제권을 부여하여 잘못된 행동을 스스로 바로잡을 수 있는 방법

ex. 되돌리기 버튼

### 4. 일관성과 표준

하나의 제품이나 서비스 안에서 일관된 레이아웃, 색상 등을 유지하는 것

ex. 디자인 시스템(구글 머터리얼 디자인, 애플 휴먼 인터페이스)

### 5. 오류 방지하는 디자인

사용자가 실수를 하지 않도록 미연에 방지하는 설계

### 6. 기억보다 인지

사용자가 모든 정보를 기억할 필요가 없도록 맥락에 맞게 적절한 정보를 제공하는 것

### 7. 유연함과 효율성

사용자의 숙련도에 따라 여러 가지 수행 방법을 준비

- 초보자는 쉽고 직관적인 기본 방법을 많이 사용한다
- 숙련자는 효율성을 위해 더 고급 기능을 사용할 수 있다

### 8. 심미성과 간결한 디자인

시각적으로는 아름다우면서 가독성과 접근성까지 높은 디자인

### 9. 사용자의 오류 인식·진단·제거

오류가 발생했을 때 오류 정보를 사용자에게 제공하여 스스로 문제를 해결하도록 설계

### 10. 충분한 지원과 도움 문서 제공

사용자가 어려움이나 궁금증이 생길 때 다양한 방법으로 도움을 줄 수 있도록 설계

ex. 가이드 문서, faq, 툴팁, 어려운 기능 옆에 도움말 표시
