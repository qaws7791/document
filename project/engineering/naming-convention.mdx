## 네이밍 컨벤션

| 분류              | 변수 종류                  | 케이스                   | 예시                           |
| ----------------- | -------------------------- | ------------------------ | ------------------------------ |
| 자바스크립트      | 상수                       | 대문자 스네이크          | MAX_LENGTH                     |
|                   | 객체, 변수, 함수, 인스턴스 | 카멜                     | maxLength, fetchPost, user     |
|                   | 클래스, 생성자 함수        | 파스칼                   | Student                        |
|                   | 모듈                       | 카멜                     | userService                    |
|                   | 이벤트 핸들러              | “on”로 시작하는 카멜     | onSubmit                       |
|                   |                            |                          |                                |
| 리액트            | 폴더, 파일                 | 케밥                     | /user-components               |
|                   | 컴포넌트 폴더              | 파스칼                   | /ProductList                   |
|                   | 컴포넌트 파일              | .jsx / .tsx 확장자       | index.tsx, ProductList.tsx     |
|                   | 컴포넌트                   | 파스칼                   | ProductList                    |
|                   | Props                      | 카멜                     | userName                       |
|                   | State                      | ”set” + 카멜             | [count,setCount] = useState(0) |
|                   | 이벤트 핸들러              | “handle”로 시작하는 카멜 | handleClick                    |
|                   | 이벤트 프로퍼티            | “on”으로 시작으로 카멜   | onClick                        |
|                   | hook                       | “use”로 시작하는 카멜    | useState, useEffect, useUser   |
|                   |                            |                          |                                |
| PostgreSQL        | 테이블                     | 스네이크 + 복수형        | user_accounts                  |
|                   | 열                         | 스네이크                 | first_name                     |
|                   |                            |                          |                                |
| Rest api          | 리소스                     | 명사형 /  케밥           |                                |
| ”_” 대신 “-” 사용 |                            |                          |                                |
| 복수형은 복수형   | …com/user-profile          |                          |                                |
| …com/items        |                            |                          |                                |
|                   | 결과 데이터 변수           | 스네이크                 | item_list                      |
|                   |                            |                          |                                |
| JSON              | 모든 키                    | 카멜                     | camel                          |
